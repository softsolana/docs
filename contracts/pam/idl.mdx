title: "IDL"
description: "Anchor IDL and TypeScript types for the PAM program."
---

Use the JSON IDL with `@coral-xyz/anchor` to construct a typed program client, or copy the TypeScript type file directly into your project.

```typescript
import { Program } from "@coral-xyz/anchor";
import type { SoftsolPam } from "./types/softsol_pam";
import idl from "./idl/softsol_pam.json";

const program = new Program<SoftsolPam>(idl as SoftsolPam, { connection });
```
