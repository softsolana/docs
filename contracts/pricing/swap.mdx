---
title: "Swap"
description: "Buy outcome shares with USDC or sell shares back for USDC."
---

`swap` lets a user buy or sell shares in a specific outcome on an open market. Prices are set by the LSMR pricing model and update with each trade.

Use [`get_quote`](/contracts/pricing/get-quote) first to preview the trade and determine a safe `slippage_bps`.

## Parameters

<ParamField body="id" type="string" required>
  The market identifier used when the market was published.
</ParamField>

<ParamField body="outcome_index" type="u8" required>
  Index of the outcome to buy or sell. Must be `1` or higher — outcome `0` (Inconclusive) cannot be traded.
</ParamField>

<ParamField body="amount" type="u64" required>
  USDC to spend in base units when buying, or number of shares to sell when selling. Must be greater than zero.
</ParamField>

<ParamField body="slippage_bps" type="u64" required>
  Minimum acceptable payout ratio in basis points. For buys, the transaction fails if `shares / amount` in bps falls below this value. For sells, it fails if `usdc_out / existing_stake` in bps falls below this value. Use `get_quote` to determine a safe value.
</ParamField>

<ParamField body="is_sell" type="bool" required>
  - `false` — buy: spend `amount` USDC to receive outcome shares.
  - `true` — sell: return `amount` shares to receive USDC.
</ParamField>

| `market` | ✓ | Market PDA. |

## Errors

| Error | Cause |
|---|---|
| `TradingInactive` | The market has already been resolved. |
| `InvalidOutcome` | `outcome_index` is `0` or out of range. |
| `InvalidAmount` | `amount` is zero, or selling more shares than held. |
| `SlippageExceeded` | Payout ratio fell below `slippage_bps`. |
| `MathOverflow` | Arithmetic overflow in share or payout calculation. |

## Example

```typescript
import * as anchor from "@coral-xyz/anchor";

const PAM_PROGRAM_ID = new anchor.web3.PublicKey("soft4qNbN2X9oykC34VZKh1VA6dkKJJNjSeko1SzAWS");
const OPERATOR = new anchor.web3.PublicKey("cxa3sYKNMk5L8PvDqEzWWLAfd4hr88d6U25W6Yz3FiE");
const id = "world-cup-final-2026";

// market is auto-derived; we need its key to derive the PAM PDAs
const [market] = anchor.web3.PublicKey.findProgramAddressSync(
  [Buffer.from("ss_market"), platform.publicKey.toBuffer(), Buffer.from(id)],
  program.programId
);
const marketSeed = market.toBuffer().subarray(0, 8);

const [resolution] = anchor.web3.PublicKey.findProgramAddressSync(
  [Buffer.from("ss_resolution"), platform.publicKey.toBuffer(), marketSeed],
  PAM_PROGRAM_ID
);
const [betEscrow] = anchor.web3.PublicKey.findProgramAddressSync(
  [Buffer.from("ss_escrow"), platform.publicKey.toBuffer(), user.publicKey.toBuffer(), marketSeed],
  PAM_PROGRAM_ID
);
const [bet] = anchor.web3.PublicKey.findProgramAddressSync(
  [Buffer.from("ss_bet"), platform.publicKey.toBuffer(), user.publicKey.toBuffer(), marketSeed],
  PAM_PROGRAM_ID
);

// Buy 10 USDC of outcome 1, minimum 95% payout ratio
await program.methods
  .swap(id, 1, new anchor.BN(10_000_000), new anchor.BN(9_500), false)
  .accounts({
    platform: platform.publicKey,
    operator: OPERATOR,
    user: user.publicKey,
    betEscrow,
    bet,
    resolution,
  })
  .signers([operator, user])
  .rpc();
```
