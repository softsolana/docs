---
title: "Get quote"
description: "Simulate a buy or sell to preview the outcome before submitting a swap."
---

`get_quote` is a read-only instruction that simulates a [`swap`](/contracts/predict/swap) and returns pricing information without modifying any state. Call this before submitting a swap to determine a safe `slippage_limit`.

## Parameters

<ParamField body="option_identifier" type="string" required>
  The market identifier used when the market was published.
</ParamField>

<ParamField body="outcome_index" type="u8" required>
  Index of the outcome to quote. Must be `1` or higher — outcome `0` (Inconclusive) cannot be traded.
</ParamField>

<ParamField body="is_buy" type="bool" required>
  - `true` — simulate buying shares with USDC. `amount` is the total USDC to spend including fees.
  - `false` — simulate selling shares for USDC. `amount` is the number of shares to sell.
</ParamField>

<ParamField body="amount" type="u64" required>
  USDC to spend in base units when buying, or number of shares when selling. Must be greater than zero.
</ParamField>

## Response

```typescript
type QuoteResult = {
  amount_out: u64;           // Shares received (buy) or net USDC received (sell)
  price_per_share: u64;      // Effective price per share, scaled ×1,000,000
  fees: u64;                 // Fee amount in USDC base units
  potential_payout: u64;     // USDC received if this outcome wins (buy only; 0 for sell)
  current_probability: u64;  // Current implied probability of the outcome, scaled ×1,000,000
  new_probability: u64;      // Implied probability after the trade, scaled ×1,000,000
};
```

<Note>
Probability values are scaled by 1,000,000. A probability of 0.65 (65%) is returned as `650_000`.
</Note>
